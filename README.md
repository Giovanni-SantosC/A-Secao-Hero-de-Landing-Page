# A-Secao-Hero-de-Landing-Page
O Flexbox é usado no .hero-section para centralizar o conteúdo vertical e horizontalmente. O .button-flexbox alinha os dois botões Call-to-Action (CTA) com gap para um espaçamento consistente. O design inclui um botão principal em destaque e um segundo botão transparente com borda, ambos com efeitos :hover sutis.
